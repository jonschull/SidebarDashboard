Welcome to the Sidebar Dashboard

This is a sample content page that demonstrates how the sidebar dashboard works. The sidebar remains visible on the left while content loads in this iframe.

External links will open in a separate browser window positioned to the right of the sidebar.

How It Works

This dashboard uses a Flask web application to:

  • Serve the sidebar interface
  • Display local content in an iframe
  • Control external browser windows for services like Gmail and Airtable
  • Position these windows to create a seamless experience

Next Steps

Try clicking on the external links (Gmail, Airtable, Google Groups) to see how they open in a separate window positioned to the right of the sidebar.
